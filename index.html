Projects

AniDisc

A movie X forum like app. Users can search filter animes that they would like to discuss. Under each anime, they can make a post and other users can comment under these posts. Users can also view their own profile or other user's profile by clicking on their profile picture. Home page displays currently airing anime, along with other popular animes.

Gashapon

A Gacha simulator, similar to a loot box if you are unfamiliar with gacha. Currently there is a JSON file of 50 characters from different series. Users can do a 10x summon or a single summon and get characters. There are 3 different rarity of characters going from 3* to 5* with 5* being the rarest, but there is a pity system implemented so users will eventually get a 5 star if they continue pulling.

Daily Tenant

A project revolving around Landlords and Tenants. A user can register an account and, on the homepage, join a building as a tenant, or create a new building and be the landlord. The landlord can make new post to inform tenants of anything. Tenants can make a request for things like repairs which the landlord can then schedule on their side. The landlord and tenants can also have a conversation on what to do about the repair.

Pokedex

A pokedex app. Users can search for pokemon by region, types, and name. On click, each pokemon will display their specific info along with their evolution chain.

Weather

A simple weather app using the ejs template. Users can search for the city that they want the weather for. Users can switch between Fahrenheit and Celsius and see the forecast for the next 5 days. Plans to include a geolocation so that it will automatically display the weather in the user's current location
